# Kazantseva-System-Module

## Назначение модуля
Модуль согласования статусов документов в системе документооборота. Позволяет отслеживать жизненный цикл документа и управлять статусами.

## Используемый стек технологий
- Python 3.11
- FastAPI
- PostgreSQL
- MIT License

## Структура
- `/docs` - сценарии Gherkin и макеты из Figma
- `/src` - спецификации данных

## Статусы документов
Черновик → На согласовании → Согласован → Подписан
(также возможен статус Отклонён)
